# üöÄ Quick Deployment Checklist

Use this checklist when deploying your app:

## Before Deployment

- [ ] Code is tested and working locally
- [ ] All sensitive data removed from code
- [ ] `.gitignore` is properly configured
- [ ] `.env` file is NOT committed to GitHub
- [ ] Requirements.txt is up to date
- [ ] You have your OpenAI API key
- [ ] You have Gmail App Password ready

## GitHub Setup

- [ ] Create GitHub repository
- [ ] Push code to GitHub
  ```bash
  git init
  git add .
  git commit -m "Initial commit"
  git branch -M main
  git remote add origin https://github.com/YOUR_USERNAME/birthday-app.git
  git push -u origin main
  ```

## Deployment Platform (Choose One)

### ‚úÖ Render (Recommended - Free)

- [ ] Wait for automatic deployment

- [ ] Initialize database: `python init_db.py`

### ‚úÖ Railway (Alternative - Free)
- [ ] Sign up at railway.app
- [ ] Deploy from GitHub
- [ ] Add PostgreSQL database
- [ ] Set environment variables
- [ ] Generate domain
- [ ] Initialize database
- [ ] Test your app

### ‚úÖ Heroku (Paid)
- [ ] Install Heroku CLI
- [ ] Create app: `heroku create`
- [ ] Add PostgreSQL: `heroku addons:create heroku-postgresql:mini`
- [ ] Set environment variables with `heroku config:set`
- [ ] Deploy: `git push heroku main`
- [ ] Initialize database: `heroku run python init_db.py`

## Environment Variables to Set

Copy these and fill in your values:

```bash
FLASK_ENV=production
SECRET_KEY=<auto-generated-or-random-string>
DATABASE_URL=<auto-set-by-platform>

OPENAI_API_KEY=<your-openai-key>

SMTP_SERVER=smtp.gmail.com
SMTP_PORT=587
SMTP_EMAIL=<your-email@gmail.com>
SMTP_PASSWORD=<your-gmail-app-password>

APP_URL=<your-app-url>
```
